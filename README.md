# Currency Converter

**Автор:** [Артём Середкин]
**Описание:** Приложение для конвертации валют с графическим интерфейсом (Tkinter), использованием внешнего API (exchangerate-api.com), сохранением истории операций в JSON и интеграцией с Git.

## Как получить API-ключ

1. Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com/).
2. Зарегистрируйтесь и получите бесплатный API-ключ.
3. Вставьте ключ в переменную `API_KEY` в файле `app.py`.

## Установка и запуск

1. Установите Python (>=3.7).
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```bash
   python app.py
   ```
4. Используйте интерфейс для конвертации валют.
5. История операций сохраняется в файл `history.json`.
6. История автоматически загружается при запуске приложения.

## Примеры использования

- Введите сумму и выберите валюты.
- Нажмите «Конвертировать».
- Проверьте таблицу истории под кнопкой.
- При ошибке ввода или API появится сообщение.
