# Currency Converter (Конвертер валют)

**Автор:** Артём Середкин

## Описание

Графическое приложение для конвертации валют. Использует внешний API для получения актуальных курсов обмена и сохраняет историю операций в локальный файл.

## Технологии

*   **Язык:** Python 3.x
*   **GUI:** Tkinter (стандартная библиотека)
*   **Сеть:** Requests (для API запросов)
*   **Хранение данных:** JSON (локальный файл)
*   **Версионирование:** Git

## Инструкция по запуску

1.  Установите Python 3.x и pip.
2.  Установите необходимые библиотеки:
    ```bash
    pip install requests tkcalendar # tkcalendar не нужен для базовой версии, но может быть полезен для расширений
    ```
3.  Получите API-ключ (см. ниже).
4.  Запустите приложение:
    ```bash
    python converter_app.py
    ```

## Как получить API-ключ

Для работы приложения требуется бесплатный или платный ключ от сервиса обмена валют.
1.  Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com/).
2.  Зарегистрируйтесь и создайте бесплатный аккаунт.
3.  В личном кабинете вы получите свой уникальный **API Key**.
4.  Откройте файл `converter_app.py` и замените строку `API_KEY = "YOUR_API_KEY"` на ваш реальный ключ.
> *⚠️ **Важно:** Не выкладывайте файл с реальным ключом в публичные репозитории. Для безопасности используйте переменные окружения.*

## Использование приложения

1.  В поле **"Из"** выберите валюту, которую хотите обменять (например, USD).
2.  В поле **"В"** выберите целевую валюту (например, EUR).
3.  Введите сумму в поле **"Сумма"** (только положительное число).
4.  Нажмите кнопку **"Конвертировать"**.
5.  Результат появится под кнопкой.
6.  Все операции автоматически сохраняются в таблицу истории внизу окна.
7.  История сохраняется в файл `conversion_history.json` и загружается при следующем запуске программы.
8.  Если ввести отрицательное число или текст, приложение покажет ошибку.
